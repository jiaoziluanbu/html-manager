# HTML 管理器 / HTML Manager

把电脑里散落各处的 `.html` 文件像文档一样集中管理。零依赖，单文件 Python 服务。

## 特性

- 🔍 **递归扫描**：自动索引 `~/` 下所有 HTML 文件，跳过 `node_modules` / `Library` / `.git` 等噪声目录
- 📂 **多种视图**：文件夹树、收藏、已删除、按标签过滤
- 🔁 **重复文件检测**：按 SHA1 hash 自动分组，显示浪费的空间
- 🏷️ **标签 + 备注**：给文件打标签、写备注，一键复用已有标签
- ⚡ **批量操作**：多选后一键批量收藏 / 打标签 / 移到废纸篓
- 👁️ **实时预览**：右侧 iframe 预览 HTML，可直接打开浏览器或在 Finder 中显示
- 🪟 **三栏可拖拽 + 折叠**：左侧栏 / 中间列表 / 右侧详情都可独立拖宽或完全折叠
- 🖱️ **右键菜单**：列表行上右键复制路径 / 文件名 / 标题等
- 🗑️ **安全删除**：走 macOS Finder 移到废纸篓，可还原

## 启动

```bash
python3 server.py
```

默认监听 `http://localhost:8765`，启动时自动打开浏览器，首次启动会自动开始扫描 `~/` 目录。

可选参数：
- `--no-open`：不自动打开浏览器
- `--no-scan`：首次启动不自动扫描

## 数据存储

- 索引数据库：`~/.html-manager/index.sqlite`
- 不复制文件本身，只保存路径、标题、大小、修改时间、SHA1、标签、备注、收藏标记

## 技术

- **后端**：Python stdlib（`http.server` + `sqlite3` + `threading`），零外部依赖
- **前端**：单文件 HTML，纯原生 JS + CSS，无构建步骤
- **平台**：macOS（"移到废纸篓"和"在 Finder 显示"使用 macOS 命令）

## 系统要求

- macOS
- Python 3.9+
